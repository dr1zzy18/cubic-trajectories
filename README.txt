Cubic Trajectory Generation Package
This package contains four nodes that generate and plot cubic polynomial trajectories using ROS.

Installation
To install this package, download or clone it into your catkin workspace and run catkin_make in the workspace directory:
"cd ~/catkin_ws/src"
"cd ar_week5_test"
"catkin_make"
"source devel/setup.sh"

Usage
To use this package, launch the cubic_traj_gen.launch file using roslaunch:
"roslaunch ar_week5_test cubic_traj_gen.launch"
This will start four nodes and a rqt_plot GUI as well as a rqt graph.
